# 3rd_year_project
Autonomous docking system for spacecraft
This repository contains all CAD and code files for the project.
The C++ code (arduino_robot_control_code_9.4.ino) is to be run on an Arduino microcontroller and the Python code (distance+pose+estimation+Arduino_v6.4.py) is intended to be run on a Raspberry Pi (recommended Raspberry Pi 4 Model B).
